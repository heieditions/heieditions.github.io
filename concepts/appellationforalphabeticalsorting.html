AppellationForAlphabeticalSorting

Bezeichnung (deutsch)
Benennung für alphabetische Sortierung
Bezeichnung (englisch)
appellation for alphabetical sorting
Definition (deutsch)
Benennung in einer Form, die für alphabetische Sortierung bestimmt ist, möglicherweise mit Umstellungen gegenüber einer natürlich angeordneten Namensform.
Definition (englisch)
An appellation given in a form intended for alphabetical sorting, possibly including transpositions in comparison with an appellation in natural order.
Anmerkung (deutsch)
In ›heiEDITIONS‹ können z. B. Personennamen in Metadaten (<persName> innerhalb des <teiHeader>) dieser Klasse zugewisen werden, um sie von einer anderen Namensform abzugrenzen, typischerweise einer Namensform, die speziell für alphabetische Sortierung vorgesehen ist.
Anmerkung (englisch)
In ›heiEDITIONS‹ this class can be assigned e. g. to personal names in metadata (<persName> inside of <teiHeader>) in order to distinguish it from another name form, typically one provided specifically for alphabetical sorting.
