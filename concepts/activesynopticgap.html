ActiveSynopticGap

Bezeichnung (deutsch)
aktive synoptische Lücke
Bezeichnung (englisch)
active synoptic gap
Definition (deutsch)
Lücke, die bei einer synoptischen Zusammenstellung mehrerer Texte in einem der Texte ausgemacht wird, um deutlich zu machen, dass im betroffenen Text tatsächlich etwas fehlt, das an dieser Stelle normalerweise erwartet würde.
Definition (englisch)
A gap which with regard to a synoptic arrangement of multiple texts has been determined in one of the texts in order to indicate that in this text something really is missing which normally would be expected to appear in this place.
Anmerkung (deutsch)
In einer synoptischen Ansicht von ›heiEDITIONS‹ wird eine aktive synoptische Lücke normalerweise immer angezeigt, unabhängig davon, ob sie gerade in der ›Fokuszeile‹ erscheint oder nicht. Sie verfügt über einen eigenen ›synoptischen Griff‹, sodass sie angeklickt werden und die Neuanordnung der Synopse auslösen kann.
Anmerkung (englisch)
An active synoptic gap is usually always displayed in an synoptic view of ›heiEDITIONS‹, whether it is appearing in the ›focus line‹ or not. It has its own ›synoptic handle‹ which allows it to be clicked on and to trigger a rearrangement of the synopsis.
