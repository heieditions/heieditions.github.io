AbstractClass

Bezeichnung (deutsch)
abstrakte Klasse
Bezeichnung (englisch)
abstract class
Definition (deutsch)
Klasse, die in ›heiEDITIONS‹ nicht zur direkten Verwendung in TEI-Dokumenten vorgesehen ist. In der Regel sind erst spezifischere Unterklassen einer abstrakten Klasse zum praktischen Einsatz bestimmt.
Definition (englisch)
A class which is not intended for direct use in ›heiEDITIONS‹ TEI documents. Usually, more specific subclasses of an abstract class are provided for practical deployment.
